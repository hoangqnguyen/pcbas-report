\begin{abstract}
We present our system for the SoccerNet Player-Centric Ball Action Spotting (PCBAS) challenge, which requires identifying \emph{what} ball-related action occurs, \emph{when} it occurs, and \emph{who} performs it across full soccer matches. We augment the provided tactical tracking data by (a) recovering missing player bounding boxes via SAM3 detection and homography projection, and (b) detecting ball positions using SAM3 with Viterbi trajectory optimization. A two-stage model---frame-level event detection followed by multi-modal classification jointly predicting action class and acting player, fusing motion, visual (DINOv2), ball proximity, and scene features through a cross-player transformer---is ensembled per-class over 25 diverse model variants with test-time augmentation, achieving a macro F1 of 0.5569 on the challenge set.
\end{abstract}
